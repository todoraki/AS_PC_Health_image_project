# Place sample / test images here

Use this folder to store leaf images for manual testing.

Example naming convention:
- `as_healthy_01.jpg`  – Acacia Senegal, healthy
- `as_unhealthy_01.jpg` – Acacia Senegal, unhealthy
- `pc_healthy_01.jpg`  – Prosopis Cineraria, healthy
- `pc_unhealthy_01.jpg` – Prosopis Cineraria, unhealthy
